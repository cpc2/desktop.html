desktop.html

Your Windows desktop, rebuilt as a website. Load a local HTML/CSS/JavaScript skin straight onto the desktop surface and make it do whatever you want.

Download for Windows View on GitHub

desktop.html is a Windows 11 runtime that hosts trusted local skins in a WebView2 surface placed behind your icons — no taskbar entry, no Alt-Tab noise. Skins talk to the OS through a promise-based window.desktop bridge: launch apps, run commands, read files, control media, and more. Multi-monitor aware, with per-monitor and spanning modes, a CLI, and automatic updates.

Heads up: skins are local programs. They can open files, run commands, and reach the network through the desktop bridge. Only install skins from sources you trust.
